SLProxy patch. Fixes a possible bug with appended acks, adds support for variable blocks to /inject, and allows more than one delegate to be installed for the same packet type.

Breaks any SLProxy application that calls RemoveDelegate or attempts to overwrite an existing delegate with a different one i.e. probably not many. Attempting to install the same delegate several times is not a problem.
